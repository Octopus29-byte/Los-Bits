## Descripción del proyecto 

Nuestro objetivo es visivilizar la realidad de México en innovación y educación, para hacer conciencia de la importancia del desarrollo de estas dos áreas dentro del país, de independizarnos tecnologicamente de otras naciones, de fomentar la innovación, de generar capital humano avanzado y de transmitir la preocupación futuros investigadores sobre este tema.
